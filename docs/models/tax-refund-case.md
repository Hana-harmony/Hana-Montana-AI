# 세무 환급 케이스 규칙

## 목적과 API

`POST /api/v1/tax/refund-status`에서 검증된 문서와 모의 거래 내역을 기준으로 미국 조세조약 환급 케이스의 샌드박스 상태를 계산한다.

## 규칙

- 모델 버전: `us-treaty-refund-case-engine-v1`
- 필수 문서 검증 완료 여부와 거래 유형, 원천징수액, 배당·매도 금액, 지분율을 입력으로 사용한다.
- 환급 가능액을 원천징수액 이하로 제한하고 국세·지방세, 선지급 수수료와 예상 지급액을 분리한다.
- 서류 미완료, 비대상 케이스, 검토 필요 상태에는 다음 조치와 위험 고지를 반환한다.
- 선지급 가능 표시는 compliance sandbox 신호이며 실제 지급을 승인하지 않는다.

## 한계

이 규칙은 운영 세무 신고, 조세조약 해석, 정부 승인, 환급금 지급·환수를 대체하지 않는다. 세율·법령·국가 범위를 확대할 때 법무 검토와 버전된 규칙·회귀 테스트가 필요하다.
